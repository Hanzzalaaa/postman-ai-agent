How it works

Three commands.
Infinite routes.

From install to a fully imported Postman collection in under a minute.

1
Install

Get the tool

$ pip install postman-agent

Single pip install. No Docker, no config files, no dependencies to manage.

2
Setup

Pick your LLM

$ postman-agent setup

Choose from 15+ LLM providers. Add your API key. Done in 30 seconds.

3
Generate

Run and import

$ postman-agent generate --scan .

AI scans every file, detects routes, groups by folder, and auto-imports into Postman.

Features

Everything you'd write
AI writes for you.

No more manual endpoints. No more guessing body shapes. No more wrong folder names.

Smart File Detection

Crawls your entire project tree and finds every route definition file automatically.

Base Path Detection

Identifies API prefixes like /api/v1 and applies them correctly across all requests.

Auto Folder Grouping

Routes are organized into logical Postman folders based on resource names and patterns.

Auth Type Detection

Detects JWT, OAuth, API Key, and Bearer patterns. Sets Postman auth headers automatically.

Smart Route Names

Human-readable names generated from route patterns — not raw endpoint strings.

Path & Query Params

Extracts :id, query strings, and optional params — adds them as Postman variables.

Realistic Request Bodies

AI infers request body shapes from validation schemas, models, and function signatures.

AI Enhancement

Goes beyond route detection — adds descriptions, example values, and pre-request scripts.

Watch Mode + Dry Run

Watch mode re-generates on file changes. Dry run previews what will be generated before import.

Usage

All CLI.
Zero config.

Terminal 
# Install
pip install postman-agent

# First-time setup (choose LLM + add API key)
postman-agent setup

# Generate from current directory
postman-agent generate --scan .

# Generate from specific folder
postman-agent generate --scan ./src/routes

# Watch mode — regenerate on file changes
postman-agent watch --scan . --interval 5

# Dry run — preview without importing
postman-agent generate --scan . --dry-run
